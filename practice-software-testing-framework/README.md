# Practice Software Testing Framework (Sprint 5)

This folder contains the Playwright TypeScript automation framework and tests aligned to Sprint 5 PRD.

## Environments
- Production UI: `https://www.practicesoftwaretesting.com`
- Local UI: `http://localhost:4200`

## Core Commands
- UI tests against production:
  - `npm run test:ui:prod`
- UI tests against local:
  - `npm run test:ui:local`
- UI smoke tests:
  - `npm run test:ui:smoke`
- UI regression tests:
  - `npm run test:ui:regression`
- API tests:
  - `npm run test:api`
